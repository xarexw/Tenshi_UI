# TENSHI Site v1.0.0

Це початковий, стабільний випуск сайту TENSHI. Ця версія встановлює основну структуру, дизайн та функціональність для публічного доступу.

## Що включено
* **Основна структура:** Реалізовано шапку (header), підвал (footer) та головну навігацію.
* **Ключові сторінки:** Створено та стилізовано головні розділи сайту (наприклад, 'Home', 'About', 'Services', 'Contact').
* **Адаптивний дизайн:** Сайт повністю адаптований для перегляду на настільних комп'ютерах, планшетах та мобільних пристроях.
* **Стилізація:** Впроваджено базовий брендинг, типографіку та стилі (CSS/SCSS).
* **Інтерактивність:** Додано базові скрипти (Vanilla JS) для інтерактивних елементів (наприклад, мобільне меню, анімації).
* **Основи SEO:** Реалізовано базові мета-теги для пошукової оптимізації.
